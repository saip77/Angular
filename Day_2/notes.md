### Understanding Angular 

#### What is Angular?

- Angular is a web application framework created and maintained by Google


#### Why Angular?

- Angular is useful for building Single Page Applications (SPA)
- Angular is built on top of TypeScript which is a superset of JavaScript that adds optional static typing to the language
- Angular CLI is a powerful tool that simplifies the process of creating and managing Angular applications
- Comes with a large ecosystem of libraries and tools that can be used to build complex applications


#### SPA vs Traditional Web Apps

> Traditional Web Apps

n a traditional web application, every time a user navigates to a new page, the browser sends a request to the server, and the server sends back a new, full HTML page. This can result in a slower user experience with a noticeable delay, as the entire page needs to reload.

> Single Page Applications (SPAs)

In contrast, a Single-Page Application loads the entire application in the initial request. As the user navigates, the application fetches only the necessary data from the server and dynamically updates the content of the current page using JavaScript, without a full page reload. This process results in a faster and more seamless user experience.
