# Setup

## Check whether you have Node.js installed

```bash
node -v
```

If you don't have Node.js installed, you can download it from [here](https://nodejs.org/en/download/).


## Install Angular CLI

```bash
npm i -g @angular/cli
```

> -g is used to install the package globally. 

> @ means that this package belongs to a specific scope.


## Check Angular CLI version

```bash
ng version
```

You should see the version of Angular CLI that you installed.


## Create a new Angular project

```bash
ng new project-name
```

This will create a new Angular project in a folder with the name `project-name`.
