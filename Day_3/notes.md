### Installing Angular

#### Install Node.js

- To check if Node.js is installed, open the terminal and type the following command:

```bash
node -v
```

#### Install Angular CLI

To install the Angular CLI, open the terminal and type the following command:

```bash
npm install -g @angular/cli
```

> Note :- 
The -g flag is used to install the package globally, which means it can be accessed from any directory in your system.


- The @ symbol at the beginning of an npm package name indicates a scoped package. It's a way of grouping related packages together under a specific namespace, which is usually the name of an organization or a user.


### Creating a New Angular Project

```
ng new my-app
```
