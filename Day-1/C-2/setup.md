# Setup

## Check whether you have Node.js installed
- To check whether you have Node.js installed, open your terminal and run the following command:

```bash
node -v
```

## Install Angular CLI
- To install Angular CLI, run the following command:

```bash
npm install -g @angular/cli
```

## Create a new Angular project
- To create a new Angular project, run the following command:

```bash
ng new angular-project
```

- This will create a new Angular project in a folder named "angular-project".
