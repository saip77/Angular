### Understanding how Angular works

- The starting point of an Angular application is the `main.ts` file, which is responsible for bootstrapping the application and creating the root component.

- The root component is the starting point of the application and is responsible for rendering the initial view of the application.

- The root component is defined in the `app.component.html` file, which is responsible for rendering the HTML structure of the application.

- The root component is defined in the `app.component.ts` file, which is responsible for creating and managing the lifecycle of the root component.
