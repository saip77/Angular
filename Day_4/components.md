# Component in Angular

## What is a Component?

- A component is a basic building block of an Angular application.
- It is a TypeScript class which has a special decorator `@Component` and it is used to define the structure and behavior of a part of the user interface.

## How to create a component?

```typescript
ng g c my-component
```

This command will create a new component called `my-component` in the `src/app` folder.
